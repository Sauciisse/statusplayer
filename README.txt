INTRODUCTION
------------

Status Player permet à n'importe qui de mettre un custom status (watching, listenning, streaming, playing) à son bot. Sans demander de connaissances spéciales en développement.
Tout est hébergé sur votre pc, donc nous recommendons de ne pas utiliser Status Player pour des serveurs de plus de 10000 membres.
Status Player V1 donne aussi accès a plusieures commandes comme (setlisten, setstream, setwatch, setgame) tout en ayant le prefix "."
Touts les membres de votre serveur peuvent avoir accès à la commande (risque d'être patch dans une version future!) donc nous recommandons de mettre le status dans un salon privé ou même un autre serveur tout dépendant vos préférences.

INSTALLATION
------------

Pour utiliser Status Player, vous devez mettre le token de votre bot dans le document token.txt que vous allez retrouver dans les fichiers que vous venez d'installer.
Vous pouvez nous contacter sur notre serveur discord si vous n'arrivez pas à setup le Status Player : https://discord.gg/r8BRvBjPEd
   
    FAQ
------------

Status Player peut-il avoir accès au notre token de notre bot?
- Non, il est impossible d'avoir accès à un fichier.txt depuis un autre ordinateur sans avoir accès à votre ordinateur.

Pourquoi quand j'ouvre Status Player, il se referme instantanément?
- Car vous avez soit entrer un mauvais token ou aucun.

Où est Status Player?
- Il est disponible sur notre serveur discord et est ouvrable en double-cliquant le bot.exe dans le fichier.
